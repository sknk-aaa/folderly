# Privacy Policy Draft

Last updated: 2026-05-21

## English

Folderly is designed to work locally on your Windows device.

### Data Collection

Folderly does not collect, transmit, sell, or share personal data.

### Local Data

Folderly stores application settings, history, and generated icon files locally on your device. This data is used only to provide app features such as applying folder icons, showing history, and reverting customized folders.

Typical local data may include:

- Folder paths customized by the user
- Source image paths selected by the user
- Generated icon files
- App settings such as language, tag names, and display options

### Images

Images selected by the user are processed locally on the device. Folderly does not upload selected images to any server.

### Analytics and Tracking

Folderly does not include analytics, advertising trackers, or telemetry.

### Microsoft Store

If Folderly is installed through Microsoft Store, Microsoft may process purchase, licensing, crash reporting, and Store-related data according to Microsoft's own policies. Folderly only uses Microsoft Store licensing APIs to check trial or purchase status.

## 日本語

Folderly は、Windows デバイス上でローカルに動作することを前提としたアプリです。

### データ収集

Folderly は、個人データを収集、送信、販売、共有しません。

### ローカルに保存されるデータ

Folderly は、アプリの設定、履歴、生成したアイコンファイルをユーザーのデバイス上に保存します。これらのデータは、フォルダアイコンの適用、履歴表示、元に戻す機能など、アプリ機能のためにのみ使用されます。

保存される可能性があるデータ:

- ユーザーがカスタマイズしたフォルダパス
- ユーザーが選択した画像ファイルのパス
- 生成されたアイコンファイル
- 言語、タグ名、表示設定などのアプリ設定

### 画像

ユーザーが選択した画像は、デバイス上でローカルに処理されます。Folderly は選択画像をサーバーにアップロードしません。

### 分析・トラッキング

Folderly には、分析、広告トラッキング、テレメトリは含まれていません。

### Microsoft Store

Folderly を Microsoft Store からインストールした場合、購入、ライセンス、クラッシュレポート、Store 関連データは Microsoft のポリシーに基づいて処理される場合があります。Folderly は、試用版または購入状態を確認するために Microsoft Store のライセンスAPIのみを使用します。
