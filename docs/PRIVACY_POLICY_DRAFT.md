## English

Folderly is designed to work locally on your Windows device.

### Data Collection

Folderly does not collect, transmit, sell, or share personal data.

### Local Data

Folderly stores application settings, history, selected image copies, and generated icon files locally on your device. This data is used only to provide app features such as applying folder icons, restoring previews, showing history, and reverting customized folders.

Typical local data may include:

- Folder paths customized by the user
- Source image paths selected by the user
- Local copies of selected images for preview restoration
- Generated icon files
- App settings such as language, tag names, and display options

### Images

Images selected by the user are processed locally on the device. Folderly does not upload selected images to any server.

### Analytics and Tracking

Folderly does not include analytics, advertising trackers, or telemetry.

### Microsoft Store

If Folderly is installed through Microsoft Store, Microsoft may process purchase, licensing, crash reporting, and Store-related data according to Microsoft's own policies. Folderly does not receive this Store account data from Microsoft.

### Contact

For support or privacy questions, contact:

Support: https://github.com/sknk-aaa/folderly/issues

## Japanese

Folderly は、Windows デバイス上でローカルに動作することを前提にしたアプリです。

### データ収集

Folderly は、個人データを収集、送信、販売、共有しません。

### ローカルに保存されるデータ

Folderly は、アプリ設定、履歴、選択した画像のコピー、生成したアイコンファイルをユーザーのデバイス上に保存します。これらのデータは、フォルダアイコンの適用、プレビューの復元、履歴表示、元に戻す機能など、アプリ機能の提供にのみ使用されます。

保存される可能性があるデータ:

- ユーザーがカスタマイズしたフォルダのパス
- ユーザーが選択した画像ファイルのパス
- プレビュー復元用に保存される選択画像のローカルコピー
- 生成されたアイコンファイル
- 言語、タグ名、表示オプションなどのアプリ設定

### 画像

ユーザーが選択した画像は、デバイス上でローカルに処理されます。Folderly は選択画像をサーバーにアップロードしません。

### 分析とトラッキング

Folderly には、分析、広告トラッカー、テレメトリは含まれていません。

### Microsoft Store

Folderly を Microsoft Store からインストールした場合、購入、ライセンス、クラッシュレポート、Store 関連データは Microsoft のポリシーに基づいて処理される場合があります。Folderly は、これらの Store アカウント情報を Microsoft から受け取りません。

### 連絡先

サポートまたはプライバシーに関する問い合わせ先:

Support: https://github.com/sknk-aaa/folderly/issues
